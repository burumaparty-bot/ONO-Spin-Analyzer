# ONO Spin Analyzer

ハンドスピナー回転計測専用プロトタイプです。

## Ver0.3.0

- `index.html` / `css/style.css` / `js/app.js` に分割
- ゲーム要素なし
- RPM、最大RPM、実FPS、信頼度、Blob面積、θ、Δθを表示・ログ化
- CSVログ、summary.json、画像ログHTMLを出力

## 公開URL

https://burumaparty-bot.github.io/ONO-Spin-Analyzer/

## 使い方

GitHubにはフォルダ構成ごとアップロードしてください。

```text
ONO-Spin-Analyzer
├── index.html
├── css/
│   └── style.css
├── js/
│   └── app.js
└── README.md
```
