# ONO Spin Analyzer Ver0.1.1-H

## 目的
Ver0.1.1で700rpm付近まで計測できた結果を基準に、800rpm到達を狙う高速許容版です。

## 変更点
- 最大有効RPMの初期値を 1500rpm に変更
- 急なRPM変化を外れ値として捨てる判定を緩和
- Blob選択時の「前回角度に近いものを優先する」重みを少し弱める
- ログ機能、CSV出力、画像保存などの負荷増加要素は追加しない

## 評価ポイント
1. 600rpm以下が従来通り安定するか
2. 700rpm付近が従来以上に安定するか
3. 800rpm付近に到達できるか
4. 明らかな誤検出やRPM飛びが増えすぎないか
