# ONO Spin Analyzer Ver0.5-Coast

## 目的
700rpm以下の実測値を信頼し、その回転ごとに惰行減速モデルを自動作成する版です。

## 基本思想
- 700rpm以下：実測ゾーン
- 700rpm超：直接計測が不安定なため、惰行モデルで推定
- モデルは固定ではなく、毎回の回転データから作成

## 実装内容
- 200〜700rpm、信頼度35%以上のデータを使用
- 約0.1秒ごとにサンプル取得
- 直近最大80点で線形減速モデルを作成
- R2 > 0.90 のときモデル有効
- 推定RPM、推定最大RPM、モデル点数、R2を表示

## 注意
まずは実験用の最小モデルです。
正確な高回転推定には、今後、二次モデルや指数モデルとの比較が必要です。
