# ONO Spin Analyzer Ver0.4-Candidate

## 目的
700rpm付近で発生する Δθ の ±150〜180°ジャンプを抑えるため、
複数Blob候補から「予測角度に近い候補」を優先する版です。

## 変更点
- 計測ループの大枠は維持
- CSV/JSON/画像ログなし
- Blob候補が複数ある場合、直近の角速度から予測角度を作り、その角度に近いBlobを優先
- ±150〜180°に飛ぶ候補には強いペナルティ
- Δθ表示は固定幅
- 解析値の表示更新は約0.5秒ごと
- 予測差(deg)を追加表示

## 見るポイント
- 700rpm付近でΔθが±150〜180°へ飛ぶ頻度が減るか
- Blob候補数が5〜7でも、現在RPMが安定するか
- 予測差が大きくなる瞬間にRPMが崩れるか
- 600rpm以下が悪化していないか
