# ONO Spin Analyzer Ver0.2-ViewSlow

## 目的
Ver0.2-View をベースに、解析値の表示更新を約0.5秒ごとに落とした版です。

## 変更点
- 計測アルゴリズムは変更なし
- CSV/JSON/画像ログなし
- Δθ、Blob候補数、半径位置、計算RPMの表示更新を約0.5秒ごとに変更
- 目視で値を追いやすくする

## 見るポイント
700rpm付近で以下を見てください。

- Δθ が 70〜90 deg/frame 程度で安定しているか
- Blob候補数が 0 になる瞬間があるか
- 半径位置が大きく乱れるか
- 計算RPMは出ているのに現在RPMが追従しないか
