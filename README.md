# ONO Spin Analyzer Ver0.2-View

## 目的
700rpm Masterをベースに、計測アルゴリズムは変更せず、画面上に軽量な解析情報だけを追加した版です。

## 追加表示
- Δθ：1フレーム間の角度変化
- Blob候補数
- 半径位置
- 計算RPM

## 重要
CSV保存、JSON保存、画像保存、ログ配列保存は追加していません。
処理負荷を極力増やさず、700rpm超で何が起きているかを確認するための版です。

## 評価ポイント
- 700rpmまでの計測性能がMasterと同等か
- 700rpmを超える直前にΔθが急に飛ぶか
- Blob候補数が0になるか、急増するか
- 半径位置が大きく乱れるか
